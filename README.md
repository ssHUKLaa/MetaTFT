# MetaTFT
# NOTE: With the recent changes to Riot's API ToS, this service is no longer feasible. It is still available to be processed locally on this source however.

A webapp used to track every game of any player in any non CN Riot region (Americas, EU, etc). A number of outputs including similarity to the mean, similarity to past performance, average rank over time, etc.

served using Django, serve locally however you would like.
